核心功能

不是泛用录屏工具，而是围绕“对局触发 + 自动录制 + 自动归档”设计的专用工作流。

自动识别对局

后台检测游戏状态，进入有效对局后自动开始录制，减少漏录与忘录。

自动结束与保存

对局结束后自动停止录制并写入文件，配合历史索引可快速定位录像。

Windows 原生录制

内置原生录制运行时，不要求用户额外安装第三方录屏软件。

音频输入可配置

支持系统音频与麦克风输入策略，可在设置页选择具体麦克风设备。

历史记录管理

支持浏览、筛选、导出和批量删除索引，快速整理本地录像资产。

中英文双语界面

应用内可切换中文与 English，满足不同玩家和团队协作场景。
