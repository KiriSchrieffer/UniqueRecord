Current Product

Minimal during play. Structured after the match.

The product is intentionally narrow today: capture the right match, save the right file, and make it manageable locally. The design leaves room for much richer AI workflows on top.

01

Automatic match detection

UniqueRecord watches game state and triggers capture only when a real match begins, reducing missed clips and unnecessary background recording.

02

Native Windows recording

The recorder runs with a bundled native runtime instead of requiring users to install extra capture tools separately.

03

Configurable capture profile

Resolution, frame rate, bitrate, container, encoder and microphone routing are handled in the app, not scattered across external software.

04

Video management built in

Local recordings are indexed for browsing, playback and cleanup, creating a stronger foundation for future AI-assisted search and editing.

05

Chinese and English UI

A bilingual interface keeps the desktop product usable for both local players and broader release audiences.

06

Installer-first distribution

Users install the desktop app directly from the official website with a standard Windows installer and persistent local data.

AI Direction

Every recording should become a usable data surface.

Text-based search

Jump directly to moments in a recording by searching what was said.

Montage generation

Combine multiple clips into a coherent highlight reel with minimal editing.
